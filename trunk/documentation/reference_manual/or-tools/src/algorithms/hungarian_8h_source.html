00016 // An O(n^4) implementation of the Kuhn-Munkres algorithm (aka the
00017 // Hungarian algorithm) for solving the assignment problem.
00018 // The assignment problem takes a set of agents, a set of tasks and a
00019 // cost associated with assigning each agent to each task and produces
00020 // an optimal (i.e. least cost) assignment of agents to tasks.
00021 // The code also enables to compute a maximum assignment by changing the
00022 // input matrix.
00023 //
00024 // This code is based on (read: translated from) the Java version
00025 // (read: translated from) the python version at
00026 //   http://www.clapper.org/software/python/munkres/
00027 // which in turn is based on
00028 //   http://www.public.iastate.edu/~ddoty/HungarianAlgorithm.html.
