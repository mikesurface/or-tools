Detailed Description

template<class T>
class operations_research::ConstIntPtrArray< T >

This class is used to store pairs of <T*, int64>.

This is useful inside constraints and expressions. The constructors, except the one with the pointer to a vector of cells will copy the data internally and will not take ownership of the data passed in argument. Its goals are:

  • to unify the construction code across the optimization libraries.
  • to provide one code to modify these mappings and apply transformations like sorting, aggregating values per pointer, or removing pointers with zero values.
